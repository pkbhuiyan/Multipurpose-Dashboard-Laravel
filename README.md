
# Multipurpose-Dashboard-Laravel
A multi purpose simple admin dashboard

- Admin and Client Dashboard
- Admin LTE3 Dashboard
- Passport and JWT
- axios
- Vue Router
- Login using social media
- ACL in Laravel
- Online user list
- Laravel+VUE Auth
- Order System
- Pusher
- Real time notification
